Native OS drag-and-drop for Daz scenes (`.duf`) and Houdini projects (`.hip`/`.hipnc`/`.hiplc`): drag a file from Explorer onto the **pane** where it's added — the whole area is the drop target, no need to aim at the Browse button, and it highlights while a supported file hovers it. Wired into the new-character scene picker and the editor's Daz scenes and Houdini projects fields. Built on Tauri's native webview drag-drop (hit-tested to the pane under the cursor), so it works with real Explorer files.
