Guide the export-directory picker to a sensible starting folder. When no export directory is set yet, the **Choose folder…** dialog now opens in the character's own folder — already inside its Houdini subfolder when that exists — so the export lands where it usually should with one click. Re-choosing an existing directory opens at the current one. You can still browse anywhere; it only changes where the dialog starts.
