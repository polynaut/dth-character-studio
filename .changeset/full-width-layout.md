Use the full app-window width on every page (desktop layout) instead of a centered narrow column. In the character editor, "Advanced workflow options" is renamed to "Advanced options" and now holds a single editable **Path in project** field — edit it to rename or reorganise a character (e.g. nest it in subfolders); collisions are rejected with a clear message.
