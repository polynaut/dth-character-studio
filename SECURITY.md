# Security Policy

## Supported Versions

Only the **latest release** is supported. The app checks for updates on
launch, and fixes ship as new releases — there are no maintenance branches
or backports.

| Version        | Supported |
| -------------- | --------- |
| latest release | ✅        |
| anything older | ❌        |

## Reporting a Vulnerability

Please **do not open a public issue** for security problems. Use GitHub's
private reporting instead: **Security → Report a vulnerability** on this
repository.

You'll get a first response within a few days. Accepted reports are fixed in
a regular release, which the built-in updater delivers to all users — you'll
be credited in the release notes unless you'd rather not be.
